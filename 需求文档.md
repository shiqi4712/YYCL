## 1. 项目背景与目标

### 1.1 背景
线上少儿编程机构的体验课教师在转化正价课过程中，频繁遭遇家长各类异议（价格、效果、时间、信任等）。目前教师沟通能力的提升主要依赖老带新和经验传授，缺乏可量化、可复盘的系统训练手段。

### 1.2 目标
构建一套 Web 端模拟训练系统，让培训师能够灵活配置训练场景，教师通过 AI 驱动的虚拟家长进行异议处理演练，并获得即时反馈与成长追踪，最终提升体验课转化率。

## 2. 角色与权限定义

| 角色 | 权限范围 | 核心操作 |
|------|---------|---------|
| **培训师（管理员）** | 全局管理权限 | 创建/管理教师账号；录入/编辑/删除训练场景与话术；查看全体教师练习数据；导出阶段性汇总报告 |
| **教师（学员）** | 个人训练权限 | 选择场景进行模拟对话；查看个人练习记录与 AI 点评；查看个人成长数据；查阅大神话术库 |

## 3. 功能需求

### 3.1 场景管理模块（培训师侧）

**需求编号：FR-001**
**需求描述：** 培训师拥有独立的场景录入与管理权限，可维护用于异议处理模拟的"训练场景"。

**功能细节：**
- **场景新建**：培训师填写场景信息并保存
  - 场景名称（如：家长质疑课程价格过高）
  - 场景分类（单选/多选：价格异议 / 效果异议 / 时间异议 / 决策异议 / 信任异议）
  - 家长画像（家长身份、孩子年龄、已知顾虑点、性格特征）
  - 初始话术（家长开口第一句话，触发对话）
  - 场景难度（初级 / 中级 / 高级）
- **场景编辑**：对已创建的场景进行修改
- **场景启停**：支持上线/下线控制，下线后教师端不可见
- **场景列表**：支持按分类、难度、创建时间筛选与搜索
- **数据校验**：场景名称唯一，必填字段完整性校验

### 3.2 模拟对话模块（教师侧）

**需求编号：FR-002**
**需求描述：** 教师登录后，浏览并选择训练场景，进入与 AI 虚拟家长的沉浸式对话。

**功能细节：**
- **场景大厅**：教师可见所有已上线的训练场景卡片，展示场景名称、分类、难度、历史练习次数
- **场景详情**：点击后查看家长画像与背景信息，确认后进入对话
- **对话交互**：
  - 教师可通过**文本输入**或**语音输入**发送消息（语音由前端 ASR 转文本后送入系统）
  - AI 虚拟家长仅通过**文字**生成回复，模拟真实家长情绪与逻辑
  - 支持多轮连续对话，单场景建议控制在 5–15 轮
  - 教师可随时点击"结束对话"终止练习
- **对话终止条件**：
  - 教师主动结束
  - AI 家长表达明确成交意向（模拟成功）
  - AI 家长明确拒绝并结束对话（模拟失败）
  - 对话轮次达到系统上限（如 20 轮）自动结束

### 3.3 AI 智能点评模块

**需求编号：FR-003**
**需求描述：** 对话结束后，系统调用 AI 引擎对整轮对话进行多维度分析，生成结构化点评报告。

**功能细节：**
- **自动触发**：对话结束后 3 秒内自动生成点评
- **点评维度**：
  - 总体评分（百分制或 1–5 星）
  - 情绪安抚能力（是否先处理情绪再讲逻辑）
  - 价值传递能力（是否清晰传达课程价值）
  - 异议处理技巧（是否使用有效话术应对核心异议）
  - 促单/收尾能力（是否把握成交信号）
  - 话术规范性（是否有不当承诺或违规表述）
- **改进建议**：针对每个维度给出具体可执行的话术优化建议
- **金牌话术推荐**：对教师表现薄弱环节，推送 1–3 条参考话术
- **教师反馈**：教师可对 AI 点评结果标记"有帮助"/"不准确"，用于后续模型优化

### 3.4 个人练习数据查阅模块（教师侧）

**需求编号：FR-004**
**需求描述：** 教师可随时查看个人历史练习记录与成长轨迹。

**功能细节：**
- **练习列表**：按时间倒序展示所有练习记录，包含场景名称、练习时间、对话轮次、总体评分
- **对话回放**：点击单条记录可完整查看当时的对话内容（教师与 AI 家长的消息流）
- **评分趋势图**：折线图展示最近 30 天内各维度评分的变化趋势
- **能力雷达图**：基于历史数据生成综合能力评估图
- **练习统计**：
  - 累计练习次数 / 时长
  - 覆盖场景数 / 分类占比
  - 平均评分 / 最高评分

### 3.5 阶段性汇总报告模块（培训师侧）

**需求编号：FR-005**
**需求描述：** 培训师可按周期查看全体教师或个人的阶段性练习汇总，识别团队共性问题与个体差异。

**功能细节：**
- **汇总周期**：支持按周 / 按月 / 自定义日期范围筛选
- **团队总览**：
  - 人均练习次数、平均评分、活跃人数
  - 各场景分类的练习热度分布（哪类异议练得最少/最多）
  - 团队能力短板排行（哪项维度平均分最低）
- **教师个人档案**：点击教师姓名查看其阶段内的详细数据（同 FR-004 的统计维度，但带时间范围）
- **共性问题挖掘**：AI 自动提炼该周期内教师群体在应对某类异议时的普遍失误点
- **导出功能**：支持导出 Excel / PDF 格式的汇总报告
- **对比分析**：可选中 2–3 位教师进行能力维度横向对比

### 3.6 大神话术库模块

**需求编号：FR-006**
**需求描述：** 系统内置销冠级应对话术库，按异议场景分类完整呈现，支持培训师维护、教师随时查阅。

**功能细节：**
- **话术浏览（教师侧）**：
  - 按异议分类（价格 / 效果 / 时间 / 决策 / 信任）浏览话术列表
  - 每条话术展示：适用场景描述、参考话术正文、话术要点拆解（为什么这样说有效）
  - 支持关键词搜索（如输入"太贵"可搜到所有价格相关话术）
  - 支持收藏常用话术至个人收藏夹
- **话术管理（培训师侧）**：
  - **新增话术**：填写分类、场景标题、参考话术、要点拆解后保存
  - **编辑话术**：修改已有话术内容
  - **删除话术**：移除不再适用的话术，删除前二次确认
  - **话术启停**：上线/下线控制，下线后教师端不可见
- **话术与场景关联**：创建训练场景时，可关联推荐 1–3 条大神话术，教师进入该场景前可直接预览

## 4. 非功能性需求

| 类别 | 需求 |
|------|------|
| 性能 | 页面首屏加载 < 2s；AI 回复延迟 < 3s；同时在线 50 人内无卡顿 |
| 安全 | 教师账号密码加密存储；敏感对话数据脱敏；培训师权限隔离 |
| 兼容 | 支持 Chrome、Edge、Safari 最新两个主版本；响应式适配 PC 与平板 |
| 可用性 | 教师端界面简洁，核心操作不超过 3 步进入对话；支持断网提示与对话恢复 |
| 扩展性 | 场景、分类、点评维度支持后台配置化扩展，无需改代码 |
| 数据 | 对话记录至少保留 12 个月；数据库定期备份 |

## 6. 附录

### 6.1 术语表

| 术语 | 解释 |
|------|------|
| 训练场景 | 一套完整的异议处理模拟设定，包含家长画像、初始话术、分类标签 |
| AI 虚拟家长 | 基于大模型扮演的家长角色，根据教师回复动态生成回应 |
| 异议分类 | 价格异议、效果异议、时间异议、决策异议、信任异议五大类 |
| 点评维度 | AI 评分所考察的具体能力项，如情绪安抚、价值传递等 |

### 6.2 已确认事项

| 事项 | 结论 |
|------|------|
| 语音对话 | 教师端支持语音输入（ASR），AI 家长仅返回文字 |
| 推送管理层 | 阶段性汇总报告不主动推送，由培训师登录系统查看 |
| SSO 接入 | 不接入机构现有系统，独立账号体系 |
| 教师账号创建 | 由培训师在后台手动创建，不支持自主注册 |
